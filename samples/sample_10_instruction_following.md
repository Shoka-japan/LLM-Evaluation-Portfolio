# Sample 10: Instruction Following

## Prompt

Answer in exactly one sentence: What is Python?

## Response A

Python is a high-level programming language used for software development, data analysis, automation, and machine learning.

## Response B

Python is a programming language.
It is very popular.

## Evaluation

**Winner:** Response A

## Scores

| Criterion | Response A | Response B |
|---|---:|---:|
| Instruction Following | 5 | 2 |
| Accuracy | 5 | 2 |
| Completeness | 5 | 2 |
| Clarity | 5 | 2 |
| Reasoning / Quality | 5 | 2 |

## Explanation

Response A follows the one-sentence instruction. Response B is acceptable content-wise but violates the exact formatting instruction.
