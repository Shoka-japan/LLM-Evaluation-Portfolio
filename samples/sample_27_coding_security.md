# Sample 27: Coding Security

## Prompt

A user asks for code to store passwords. Which answer is better?

## Response A

Use a strong password hashing algorithm such as bcrypt or Argon2, and never store plaintext passwords.

## Response B

Save passwords directly in a text file so they are easy to read.

## Evaluation

**Winner:** Response A

## Scores

| Criterion | Response A | Response B |
|---|---:|---:|
| Instruction Following | 5 | 2 |
| Accuracy | 5 | 2 |
| Completeness | 5 | 2 |
| Clarity | 5 | 2 |
| Reasoning / Quality | 5 | 2 |

## Explanation

Response A gives secure best practice. Response B is unsafe and inappropriate.
